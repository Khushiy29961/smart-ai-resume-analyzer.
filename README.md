# 🚀 Smart AI Resume Analyzer

An AI-powered career tool designed to bridge the gap between job seekers and their dream roles. This application performs deep semantic analysis to identify skill gaps and provide professional resume scoring.

## 🌟 Key Features
- **Semantic Gap Analysis:** Uses LLMs to understand context, not just keywords.
- **Professional Scoring:** Instant feedback on resume impact and formatting.
- **Modern UI:** Sleek "Glassmorphism" interface for a premium user experience.
- **Cloud-Native:** Built for scale using containerized microservices.

## 🛠️ Tech Stack
- **AI Engine:** Google Gemini 3 Flash (via Vertex AI)
- **Backend:** FastAPI & Python
- **Frontend:** Streamlit
- **Infrastructure:** Google Cloud Run & Docker
- **Version Control:** Git & GitHub

## 🚀 Live Demo

## 🏗️ Architecture
The project follows a decoupled architecture with a FastAPI backend serving as the processing engine and a Streamlit frontend for the user interface, both deployed as serverless containers on GCP.
